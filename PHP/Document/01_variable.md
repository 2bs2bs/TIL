# 変数

## PHPでの変数宣言の仕方

  ```
    $変数名
  ```

## 変数の更新

  ```
    $x = $x + 1; → $x += 1;
  ```

  追加するの1のときだけ更に省略可
  ```
    $x += 1; → $x ++;
    $x -= 1; → $x --;
  ```
  ※表現するとき注意しないといけないぽい
  ```
    $x = 3;
    $y = 3;

    echo ++$x; => 4 echoの前に+1される
    echo $y++; => 3 echoの後に+1される
  ```

## 文字列連携
  ```
  $lesson = 'PHP'
  echo $lesson.'入門';  => // 'PHP入門'

  $level = '入門';
  echo $lesson.$level;  => // 'PHP入門'
  ```

  応用的な代入の仕方
  ```
  $name = '忍者';
  $name .= 'わんこ';  // $name = $name.'わんこ' 最初の$nameには忍者が格納

  echo $name; // 忍者わんこ
  ```

## 変数展開法
  ""で囲って{$変数名}
  ```
    $name = baqwo;
    echo "こんにちは{$name}"; // こんにちはbaqwo
  ```
