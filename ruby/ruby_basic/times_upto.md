# timesメソッドとuptoメソッドの使い方

## timesメソッド

`times`メソッドは指定した回数繰り返したい時に使用する

```

オブジェクト.times do |変数|
	処理１
	処理2
end

```
do~endを省略した書き方
```

オブジェクト.times{|変数|
	実行する処理
	実行する処理
}

```

## uptoメソッド
開始数〜最大数まで１ずつ増加しながら繰り返しを行う。
```
オブジェクト.upto(max){|変数|
	処理１
	処理2
}
```

例
```
3.upto(7){|num|
	num += num.to_s
	puts num
}
```
