Class とは
  - オブジェクトの設計図
  - classから作られたオブジェクトはインスタンスと呼ばれる
  - 似たようなオブジェクトは複数作成するときに便利

オブジェクト思考プログラミングとは(OOPとは)
  - 現実世界のものに見立ててプログラムする方法
  - どのようにアプリケーションを作成するかという方法の一つ
  - 人間にとってわかりやすくロジックを分割する方法の一つ
