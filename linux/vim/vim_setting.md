# 概要
vimの設定方法について

# 設定方法
`$HOME/.vimrc`ファイルを作成し、設定内容を記載することで適応できる

# 一例
- 挿入モードであるときにそのモードであることがわかるようにする
```
set showmode
```
- 検索単語を強調表示しないように修正
```
set search
```
- 自動インデント
```
set autoindent
```
- タブ数の設定
```
set tabstop=4
```
- タブをスペースに変換
```
set expandtab
```
- 構文の強調表示をオンにする
注意事項: setを使用しないので注意
```
syntax on
```
