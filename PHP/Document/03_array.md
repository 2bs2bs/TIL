##  配列の基礎構文
  ```
  $配列名 = array(値1, 値2, 値3, ...)
  ```
  先頭から0,1,2となる。

  ### 取り出し方
    '''
    $配列名[インデックス番号]
    '''
    ```
    $names = array('aaa', 'bbb', 'ccc')

    echo $names[0] // 'aaa'
    echo $names[1] // 'bbb'
    ```
  
  ### 値の追加, 更新
    末尾への追加
    ```
    $配列名[] = 値
    ```
    値の更新
    ```
    $配列名[1] = 値
    ```

##  連想配列
  ```
  $連想配列名 = array(
    'key1' => 'value1', 
    'key2' => 2,
    'key3' => 'value3'
  )
  ```
  `,`で区切る
  `=>`でkeyとvalueを連結

  ### 取り出し方
  ```
  $連想配列名[key]
  ```

  ### 追加の仕方
  ```
  $連想配列名[new_key(追加するkey)] = 'new_value'(追加するvalue);
  ```
